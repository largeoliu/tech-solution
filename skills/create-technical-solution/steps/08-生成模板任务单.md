# 步骤 8：生成模板任务单

## 输入
- 当前模板
- 共享上下文清单
- 状态文件

## 操作
1. 为每个槽位明确：槽位标识、槽位语义、可填写边界、必须消费的共享上下文、参与专家、每位专家必答问题、禁止越界项、依赖槽位、缺失即停止的上下文、缺口或阻塞项
2. 严格按当前模板顺序列出槽位
3. 将 WD-TASK 区块写入 working draft
4. 向用户展示摘要

## 完成标准
- 当前模板每个槽位都有对应任务单条目
- 所有必填字段完整
- WD-TASK 区块已写入 working draft
- 摘要已向用户展示

## 输出
- 更新状态文件 checkpoints.step-8
- WD-TASK 写入 working draft

## 门控
模板任务单未完成时不能组织专家分析，不得进入第 9 步。

## 回退信号
模板槽位定义、槽位边界、参与专家分配、阻塞条件变化。
