# 步骤 2：检查 .architecture 前置文件

## 输入
- 状态文件

## 操作
1. 确认 `.architecture/members.yml` 存在且可读
2. 确认 `.architecture/principles.md` 存在且可读
3. 确认 `.architecture/templates/technical-solution-template.md` 存在且可读
4. 任一缺失时停在第 2 步并给出缺失清单，引导用户使用 `bootstrap-architecture`

## 完成标准
- 三文件存在且可读
- 任一缺失时停在第 2 步并给出缺失清单

## 输出
- 更新状态文件 checkpoints.step-2

## 门控
三文件任一缺失时立即停止，不得进入第 3 步。

## 回退信号
前置文件被删除或移动。
