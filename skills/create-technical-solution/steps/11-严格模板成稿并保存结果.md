# 步骤 11：严格模板成稿并保存结果

## 输入
- 当前生效模板
- 协作收敛纪要
- working draft
- 状态文件

## 操作
1. 以模板为底稿，从 SLOT-01 开始逐槽填充
2. 每个槽位只能写入 WD-SYN 中该槽位的收敛结论
3. 无收敛内容时保留占位说明
4. 创建缺失目录（`.architecture/technical-solutions/working-drafts/` 或 `.architecture/technical-solutions/`）
5. 写入最终文档到 `.architecture/technical-solutions/[slug].md`
6. 如文件已存在，确认覆盖或另存

## 完成标准
- 一级章节数量和顺序与模板完全一致
- 所有已收敛内容都找到语义归宿
- 无回退到默认模板痕迹
- 文件已写入目标目录
- 如文件已存在用户已确认覆盖或另存

## 输出
- 更新状态文件 checkpoints.step-11
- 最终文档写入 `.architecture/technical-solutions/[slug].md`

## 门控
最终成稿未完成时不能执行吸收检查，不得进入第 12 步。

## 回退信号
只影响最终模板落位。
